## Описание проекта
Этот проект представляет собой базовую реализацию структуры для интернет-магазина, используя объектно-ориентированное программирование (ООП). В рамках проекта разработаны классы для работы с категориями и продуктами, что позволяет в дальнейшем расширять функциональность интернет-магазина.

## Структура проекта
* main.py - основной файл, содержащий классы и логику работы.
* products.json - файл с примером данных для подгрузки категорий и продуктов.
* tests/ - директория, содержащая тесты для проверки работы классов с использованием pytest..
## Тестирование
Для запуска тестов выполните команду: pytest tests
Тесты проверяют:
* Корректность инициализации объектов класса Product и Category.
* Подсчет количества категорий и продуктов.
## Дополнительное задание
В рамках дополнительного задания реализована функция подгрузки данных о категориях и продуктах из файла products.json`. Функция читает файл в формате JSON и создает объекты классов на основе данных.

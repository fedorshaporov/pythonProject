## Описание проекта
Этот проект представляет собой базовую реализацию структуры для интернет-магазина, используя объектно-ориентированное программирование (ООП). В рамках проекта разработаны классы для работы с категориями и продуктами, что позволяет в дальнейшем расширять функциональность интернет-магазина.

## Структура проекта
### main.py 
основной файл, содержащий классы и логику работы.
- **BaseProduct** - базовый абстрактный класс, который является родительским для классов продуктов.
- **Product** - базовый класс для всех продуктов, содержащий общие атрибуты и методы.
- **Category** - класс, представляющий категорию, в которую входят различные продукты. Управляет объектами Product.
- **Класс Smartphone** наследуется от Product и включает дополнительные атрибуты: efficiency, model, memory, color,
- **Класс LawnGrass** также наследуется от Product и включает атрибуты: country, germination_period, color.
### tests/ 
директория, содержащая тесты для проверки работы классов с использованием pytest..
## Тестирование
Для запуска тестов выполните команду: pytest tests
Тесты проверяют:
* Корректность инициализации объектов класса Product и Category.
* Подсчет количества категорий и продуктов.
